# Convex-Concave 0-Sum Stochastic Stackelberg Games


### Environments
Environments are found in gym_examples/envs/

Dependencies are
```
jax
optax
dm-haiku
gym
pandas
matplotlib
seaborn
numpy
tensorboard
imageio
```

Pythoin version 3.9.2

## Running the script
to run 
cd into `gym_examples/gym_examples/src`
```
python run_script.py
```
